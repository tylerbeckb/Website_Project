Fundraising is such an important way to help support change and keep the charities fighting for what is right.

Run a Marathon

  • • Pick a Marathon

    • Go to find a race for all marathons running each year
  • • Train

    • - Create a training plan for 24 weeks
    - Work yourself up by running little and often
    - Pace yourself
    - Take rest days
  • • Set up Fundraising Page

    • - Use GoFundMe, set up a page using the steps on the website
    - Use Facebook, they have a good fundraiser page which can be easily shared with your following
    - Share the pages across all platforms and tell people about the page
  • • Run

    • - Have fun and give it your best shot!!
